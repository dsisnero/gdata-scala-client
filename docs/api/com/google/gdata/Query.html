class Query(val searchQuery : SearchQuery)
extends AnyRef
Base class for GData queries. It has a category part and a search query part. Categories are added using the '/' operator. The search query is added using 'suchThat'. Retrieve the URL by calling 'mkUrl(base)'. It returns a URL looking like 'base/-/categories/../lastCategory?q='. It takes care of URL encoding. It handles the default GData parameters (author, alt, updated-min, updated-max, published-min, published-max, start-index, max-results). @see GData queries. Subclasses should override 'searchName' to change 'q' to something else. New parameters should be implemented as methods in terms of 'addParam'.
