GData Scala Client

This document is the API specification for GData Scala Client Library Documentation

Class Summary
trait AclEntries extends AtomEntries
Extends atom entries with access control elements, used by calendar.
case class AclScope (val tpe : java.lang.String, val value : scala.Option[java.lang.String]) extends scala.Product
An access control list scope. Type can be 'user', 'domain' or 'default'. Value can be an email address or a domain name.
trait CalendarEntries extends AtomEntries
Entries in a calendar's feed. Each entry describes a calendar.
class CalendarQuery extends Query
A calendar query. It adds methods for specifying whether recurring events should be expanded, which events should be returned, the timezone for the returned times, etc.
class CalendarService (appName : java.lang.String) extends Service
A service class for connecting to the calendar service. It provides methods for getting a user's calendars, managing events and calendar subscriptions.
class StdAclFeed extends AtomFeeds with AclEntries
A standard access control list feed.
class StdCalendarsFeed extends AtomFeeds with CalendarEntries
class StdEventsFeed extends AtomFeeds with EventEntries
A standard event feed.
