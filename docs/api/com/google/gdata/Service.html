abstract class Service(appName : java.lang.String, service : java.lang.String)
extends AnyRef
A base class for Google services. It provides the basic querying mechanism and authentication. It caches 'gsessionid' parameters when redirected, and ships them with every future query. All Google services should extend this class and provide specific methods for that service. If a specific class is not yet available, this class can be used with generic StdAtomFeed to retrieve and further process feeds.
Parameters
appName - The application name. It should contain the company name, application name and its version, separated by '-'.
service - The service name. See
