GData Scala Client

This document is the API specification for GData Scala Client Library Documentation

Class Summary
trait AtomEntries extends Entries
Atom Entries provides a type of Atom entries and a pickler for the AtomEntry. The Entry type remains abstract, to allow further refinements.
trait AtomFeeds extends Feeds
Atom feeds refines Feeds with Atom-like feeds.
case class Category (val term : java.lang.String, val scheme : scala.Option[java.lang.String], val label : scala.Option[java.lang.String]) extends scala.Product
An Atom category, as defined by the Atom spec.
abstract class Content (tpe : java.lang.String) extends AnyRef
A content element as defined by the Atom syndication format.
trait Entries extends AnyRef
An interface for abstract entries. It is further refined by providing lower bounds for the Entry type. @see AtomEntries.
trait Feeds extends AnyRef
A module for Feeds. Use this by mixing it in a class that has an Entries implementation. This module provides a feed and a feed pickler. This module requires an entry type and an entry pickler.
case class Generator (val name : java.lang.String, val uri : scala.Option[java.lang.String], val version : scala.Option[java.lang.String]) extends scala.Product
An Atom generator construct, as defined by the Atom spec.
case class HtmlContent (val text : java.lang.String) extends Content with scala.Product
An Html content.
case class Link (val href : java.lang.String, val rel : scala.Option[java.lang.String], val tpe : scala.Option[java.lang.String], val hrefLang : scala.Option[java.lang.String], val title : scala.Option[java.lang.String], val length : scala.Option[java.lang.String]) extends HasStore with scala.Product
An Atom link construct, as defined by the Atom spec.
trait LinkNavigation extends AnyRef
Mix in this trait in any container of links. It allows retrieving links based on the 'rel' attribute
case class OutOfLineContent (val src : java.lang.String, val tpe : java.lang.String) extends Content with scala.Product
An out of line content (linked to an external URL).
case class Person (val name : java.lang.String, val uri : scala.Option[java.lang.String], val email : scala.Option[java.lang.String]) extends (java.lang.String, scala.Option[java.lang.String], scala.Option[java.lang.String]) with HasStore
A person construct, according to the Atom specification.
class Source extends AnyRef
An Atom source element. It is used when entries are copied over from another feed to keep around metadata bout the source feed.
class StdAtomFeed extends AtomFeeds with AtomEntries
A standard Atom feed, with only the elements defined by Atom Syndication Format
case class Text (val tpe : scala.Option[java.lang.String], val content : java.lang.String) extends (scala.Option[java.lang.String], java.lang.String)
A text construct, according to the Atom specification. The content is uninterpreted.
case class TextContent (val text : java.lang.String) extends Content with scala.Product
A plain text content.
case class XhtmlContent (val div : scala.xml.NodeSeq) extends Content with scala.Product
A Xhtml content.
