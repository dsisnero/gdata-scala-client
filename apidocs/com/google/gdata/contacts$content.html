Class Summary
class ContactsQuery extends Query
A Contacts API query. It adds service specific query parameters, like ordering, sorting and whether deleted contacts should be included.
class ContactsService (appName : java.lang.String) extends Service
A service class for Google Contacts API. It provides conenience methods for retrieving, adding, updating and deleting contacts. The only feed described by this service is private, therefore authentication is required.
class StdContactsFeed extends AtomFeeds with ContactEntries
A standard contacts feed.
