trait AuthToken extends AnyRef
An authentication token used by GData services. It encapsulate the logic necessary to fill the 'Auth' HTTP header in GData requests. Such tokens are obtained using an AuthFactory.

class GDataRequest (method : scala.Enumeration.Value, url : java.net.URL) extends AnyRef
This class encapsulates an HTTP request to a GData service. It sets up the HTTP connection and handles authentication. This class throws IllegalStateException in exactly the same cases as HttpURLConnection.
class GDataRequestException (msg : java.lang.String) extends java.lang.Exception with AnyRef
Base class for GData request exceptions.
class HttpConnection (val underlying : java.net.HttpURLConnection) extends AnyRef
Scala wrapper over HttpURLConnection. Turns getter/setters into properties and implements a request headers as properties. Like the underlying Java connection, it supports only one request/response. The 'connect' method returns an HttpResponse that gives access to the response headers and streams. You can set headers using the following syntax: connection("Content-Type") = "text/xml" A similar syntax can be used on the HttpResponse object to retrieve response headers: val response = connection.connect log("redirected to: " + response("Location"))
