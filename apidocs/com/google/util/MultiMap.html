MultiMap

trait MultiMap [A, B, TSet <: Set[B]] extends Map[A, TSet]

This class is typically used as a mixin. It turns maps which map A to Set[B] objects into multi maps which map A to B objects. This class allows some choice of the concrete type of Set, while the standard one in the Scala library is fixing it to HashSet. Use this when your elements have no hashCode method, like NodeSeq in the XML libraries.

Adapted from the Scala standard libraries. Parked here until it moves to the standard library.
