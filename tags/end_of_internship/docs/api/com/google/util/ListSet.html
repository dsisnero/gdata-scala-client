class ListSet[A]
extends scala.collection.mutable.Set[A]
A mutable Set backed by a List. Inserts are constant time, removes and contains are linear. Use when sets are supposed to remains small, and a HashSet would not fit the bill (for instance, no implementation of hashCode).
