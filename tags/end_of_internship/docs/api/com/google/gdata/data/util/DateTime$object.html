def apply (year : Int, month : Int, day : Int, hour : Int, min : Int, sec : Int, frac : scala.Option[Double], offset : Long) : DateTime
Create a date time object with the given coordinates. The date is considered to be shifted from UTC with the amount of milliseconds given in 'offset'.

It checks that the month is between 1 and 12, day of the month between 1 and 31 and normal hour and minutes check. It doesn't do more fancy checks (check day of month w.r.t. to the given month, leap year/minute/second).
