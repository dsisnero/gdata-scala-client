GData Scala Client

This document is the API specification for GData Scala Client Library Documentation

Class Summary
trait ContactsEntries extends AtomEntries
Youtube user contact entries.
trait PlaylistEntries extends AtomEntries
Users playlist link entries. They contain a link to a video feed containing the videos in the playlist.
trait PlaylistVideoEntries extends VideoEntries
This trait refines video entries to add position and an optional description.
case class Statistics (val viewCount : scala.Option[Int], val videoWatchCount : scala.Option[Int], val subscriberCount : scala.Option[Int], val lastWebAccess : scala.Option[DateTime]) extends scala.Product
You tube statistics. Used in user profile entries.
class StdCommentsFeed extends AtomFeeds with AtomEntries
A standard comment feed, using the atom feeds.
class StdContactsFeed extends AtomFeeds with ContactsEntries
A concrete, standard contacts feed.
class StdPlaylistFeed extends AtomFeeds with MediaRss with PlaylistVideoEntries
A concrete, standard playlist feed. It feeds a playlist as a sequence of video entries, ordered by a 'position' field.
class StdSubscriptionFeed extends AtomFeeds with SubscriptionEntries
A concrete standard subscription feed.
class StdUserPlaylistsFeed extends UserPlaylistsFeed with AnyRef
A concrete, standard user playlists feed. Entries are links to a video feed for the given playlist.
class StdUserProfileEntry extends UserProfileEntries
A concrete standard user profile entry.
class StdVideoFeed extends VideoFeeds with AnyRef
A concrete, standard video feed. It turns all types in VideoFeeds to concrete types. Feed is an atom feed, entries are video entries, and the media group content has the youtube elements.
trait SubscriptionEntries extends AtomEntries
User Subscription entries extends atom entries with subscription-specific elements like 'queryString' and 'feedLink'.
trait UserPlaylistsFeed extends AtomFeeds with PlaylistEntries
User playlist feeds. It is an atom feed with playlist description entries.
trait UserProfileEntries extends AtomEntries
trait VideoEntries extends AtomEntries
A component for video entries. It requires a comments feed instance, and an implementation of MediaRss. It refines the media rss content to support youtube specific attributes.
trait VideoFeeds extends AtomFeeds with MediaRss with VideoEntries
A VideoFeed class that brings together an atom feed and youtube video entries.
class YouTubeQuery extends Query
A YouTube specific query.
class YouTubeService (appName : java.lang.String) extends Service
This class takes care of the communication with the you tube service and provides methods for retrieving and navigating between different feeds. Example: val s = new YouTubeService("mycompany-test-1.0") for (e <- s.getVideos(Query.empty / "Comedy")) println(e.rating) It will print all the ratings for videos retrieved by the given query (those having the 'Comedy' category).
