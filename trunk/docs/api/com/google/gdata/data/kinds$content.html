GData Scala Client

This document is the API specification for GData Scala Client Library Documentation

Class Summary
case class Comments [Feed](val rel : scala.Option[java.lang.String], val feedLink : scala.Option[FeedLink[Feed]]) extends scala.Product
A gd:comments element. Contains a comments feed for the enclosing entry.
trait ContactEntries extends AtomEntries
Refine Entry to the contact kind. It represents contact infromation for an entity, including emails, phone numbers, addresses etc.
case class Email (val address : java.lang.String, val label : scala.Option[java.lang.String], val rel : scala.Option[java.lang.String], val primary : Boolean) extends scala.Product
An email address as exposed by GData.
class EntryLink [Entry] extends AnyRef
Represents a logically nested entry. For example, a representing a contact might have a nested entry from a contact feed.
trait EventEntries extends AtomEntries
An event kind. It extends Atom entries with elements for describing a calendar event: attendees, location, recurrence, comments, etc.
case class ExtendedProperty (val name : java.lang.String, val value : java.lang.String) extends scala.Product
A GData extended property is a name-value pair of Strings.
class FeedLink [Feed] extends HasStore
A gd:feedLink element. It represents a logically nested feed of abstract type 'Feed'.
case class Im (val address : java.lang.String, val label : scala.Option[java.lang.String], val rel : scala.Option[java.lang.String], val protocol : scala.Option[java.lang.String], val primary : Boolean) extends scala.Product
An instant messaging address, as exposed by GData.
case class Organization (val label : scala.Option[java.lang.String], val rel : scala.Option[java.lang.String], val primary : Boolean, val name : scala.Option[java.lang.String], val title : scala.Option[java.lang.String]) extends scala.Product
An organization, typically associated with a contact.
case class OriginalEvent (val id : java.lang.String, val href : java.lang.String, val when : When) extends scala.Product
Identifies a particular instance in a recurrent event. Equivalent to Recurrence ID in RFC 2445.
case class PhoneNumber (val label : scala.Option[java.lang.String], val rel : scala.Option[java.lang.String], val uri : scala.Option[java.lang.String], val primary : Boolean, val phone : java.lang.String) extends scala.Product
A phone number as exposed by GData
case class PostalAddress (val label : scala.Option[java.lang.String], val rel : scala.Option[java.lang.String], val primary : Boolean, val address : java.lang.String) extends scala.Product
A postal address, as represented by GData.
case class Rating (val average : scala.Option[Double], val min : Int, val max : Int, val numRaters : scala.Option[Int], val rel : java.lang.String, val value : scala.Option[Int]) extends scala.Product
A gd:rating element. Represents a numeric rating of the enclosed entity.
class RecurrenceException [EventEntry <: EventEntry] extends AnyRef
An event that's an exception to a recurring event.
case class Reminder (val absoluteTime : scala.Option[DateTime], val method : scala.Option[java.lang.String], val days : scala.Option[Int], val hours : scala.Option[Int], val minutes : scala.Option[Int]) extends scala.Product
A reminder, usually found inside a 'when' element.
class StdContactEntries extends ContactEntries
A concrete class for contact entries. Useful when contacts are not part of a feed.
class When extends AnyRef
This class represents a period of time or an instant. If endTime is not specified, it is considered an instant in time when the startTime has a 'time' component, or a whole-day if startTime is only a date.
class Where [CEntry <: ContactEntry] extends AnyRef
A place (such as an event location) associated with the containing entity. The type of the association is determined by the rel attribute; the details of the location are contained in an embedded or linked-to Contact entry.
class Who [ContactEntry <: ContactEntry] extends AnyRef
A person associated to the containing entity. The relation is specified by the 'rel' field.
